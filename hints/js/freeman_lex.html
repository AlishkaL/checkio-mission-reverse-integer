Ok, what should I start from?

Find the absolute value of a given number with Math.abs() function. 

Math.abs(num)

Convert it to string with toString() method and split with split() method with empty string argument. 

Math.abs(num).toString().split('')

Reverse the array with reverse() method. 

Math.abs(num).toString().split('').reverse()

Join the array back to string with join() method with empty string argument. 

Math.abs(num).toString().split('').reverse().join('')
What to do next?

if the original number is negative, complement the expression with "-" before. For this purpose use ternary operator. 

(num < 0 ? '-':'')
I still need a hint.

Concatenate two above expression together and convert the result back to integer. 

parseInt((num < 0 ? '-':'') + Math.abs(num).toString().split('').reverse().join(''))
So, how my final solution should look like?

You just need to wrap your code into function expression and return the above expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function reverseDigits(num: number, reversed = 0, sign=1): number {

    return parseInt((num < 0 ? '-':'') + Math.abs(num).toString().split('').reverse().join(''))
}

Without using string conversion continuously divide the given number by 10 and build a new reversed number. 

function reverseDigits(num: number): number {

    let reversed = 0;
    const sign = num < 0 ? -1 : 1;
    num = Math.abs(num);
    while (num !== 0) {
        reversed = reversed * 10 + num % 10;
        num = Math.floor(num / 10);
    }
    return sign * reversed;
}

Recursive solution with two additional arguments of function. 

function reverseDigits(num: number, reversed = 0, sign=1): number {

    if (num < 0) sign = -1;
    if (num === 0) return sign * reversed;
    num = Math.abs(num);
    
    return reverseDigits(Math.floor(num / 10), reversed * 10 + num % 10, sign);
}
