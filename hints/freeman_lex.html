What to do next?

if the original number is negative, complement the expression with "-" before. For this purpose use Python ability to interpret logic value as integer. 

'-' * (num < 0)
I still need a hint.

Concatenate two above expression together and convert the result back to integer. 

int('-' * (num < 0) + str(abs(num))[::-1])
So, how my final solution should look like?

You just need to wrap your code into function expression and return the above expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def reverse_digits(num: int) -> int:

    return int('-' * (num < 0) + str(abs(num))[::-1])

Without using string conversion continuously divide the given number by 10 and build a new reversed number. 

def reverse_digits(num: int) -> int:

    reversed_num, absolute_num = 0, abs(num)
    while absolute_num:
        absolute_num, remainder = divmod(absolute_num, 10)
        reversed_num = reversed_num * 10 + remainder
        
    return reversed_num if num >= 0 else -reversed_num

Recursive solution with two additional arguments of function. 

def reverse_digits(num: int, reversed_num=0, sign=1) -> int:

    if num < 0: sign = -1
    if num == 0: return reversed_num if sign > 0 else -reversed_num
    a, b = divmod(abs(num), 10)
    
    return reverse_digits(a, reversed_num * 10 + b, sign)
