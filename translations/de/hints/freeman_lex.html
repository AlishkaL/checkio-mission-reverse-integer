Finden Sie den absoluten Wert einer gegebenen Zahl mit der Funktion abs(). 

abs(num)

Konvertieren Sie sie mit der Funktion str() in eine Zeichenkette. 

str(abs(num))

Kehren Sie die Zeichenkette mit negative slicing um. 

str(abs(num))[::-1]
Was ist als nächstes zu tun?

wenn die ursprüngliche Zahl negativ ist, ergänzen Sie den Ausdruck vorher mit "-". Verwenden Sie zu diesem Zweck die Fähigkeit von Python, logische Werte als Ganzzahlen zu interpretieren.

Verketten Sie die beiden obigen Ausdrücke miteinander und konvertieren Sie das Ergebnis in eine Ganzzahl.

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur Ihren Code in einen Funktionsausdruck einpacken und den obigen Ausdruck zurückgeben. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def reverse_digits(num: int) -> int:

    return int('-' * (num < 0) + str(abs(num))[::-1])

Dividieren Sie die angegebene Zahl kontinuierlich durch 10 und bilden Sie eine neue umgekehrte Zahl, ohne die Stringkonvertierung zu verwenden. 

def reverse_digits(num: int) -> int:

    reversed_num, absolute_num = 0, abs(num)
    while absolute_num:
        absolute_num, remainder = divmod(absolute_num, 10)
        reversed_num = reversed_num * 10 + remainder
        
    return reversed_num if num>= 0 else -reversed_num

Rekursive Lösung mit zwei zusätzlichen Argumenten der Funktion. 

def reverse_digits(num: int, reversed_num=0, sign=1) -> int:

    if num < 0: sign = -1
    if num == 0: return reversed_num if sign> 0 else -reversed_num
    a, b = divmod(abs(num), 10)
    
    return reverse_digits(a, reversed_num * 10 + b, sign)
