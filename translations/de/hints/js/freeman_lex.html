Fangen Sie eine Idee 💡! Finden Sie den absoluten Wert einer gegebenen Zahl, wandeln Sie ihn in einen String um und kehren Sie ihn um. Wenn die ursprüngliche Zahl negativ ist - ergänzen Sie die endgültige Zahl mit Minus. Konvertieren Sie alles zurück in eine ganze Zahl.

Ok, womit soll ich anfangen?

Finden Sie den absoluten Wert einer gegebenen Zahl mit der Funktion Math.abs(). 

Math.abs(num)

Konvertieren Sie sie mit der Methode toString() in eine Zeichenkette und splitten Sie sie mit der Methode split() mit einem leeren String-Argument. 

Math.abs(num).toString().split('')

Kehren Sie das Array mit der Methode reverse() um. 

Math.abs(num).toString().split('').reverse()

Fügen Sie das Array mit der Methode join() mit einem leeren String-Argument wieder zu einem String zusammen. 

Math.abs(num).toString().split('').reverse().join('')
Was ist als nächstes zu tun?

wenn die ursprüngliche Zahl negativ ist, ergänzen Sie den Ausdruck vorher mit "-". Verwenden Sie zu diesem Zweck den ternären Operator. 

(num < 0? '-':'')
Ich brauche noch einen Hinweis.

Verketten Sie die beiden obigen Ausdrücke miteinander und konvertieren Sie das Ergebnis in eine Ganzzahl. 

parseInt((num < 0? '-':'') + Math.abs(num).toString().split('').reverse().join(''))
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur Ihren Code in einen Funktionsausdruck einpacken und den obigen Ausdruck zurückgeben. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

function reverseDigits(num: number, reversed = 0, sign=1): number {

    return parseInt((num < 0? '-':'') + Math.abs(num).toString().split('').reverse().join(''))
}

Dividieren Sie die angegebene Zahl kontinuierlich durch 10 und bilden Sie eine neue umgekehrte Zahl, ohne die Stringkonvertierung zu verwenden. 

function reverseDigits(num: number): number {

    let reversed = 0;
    const sign = num < 0? -1: 1;
    num = Math.abs(num);
    while (num!== 0) {
        reversed = reversed * 10 + num % 10;
        num = Math.floor(num / 10);
    }
    return sign * reversed;
}

Rekursive Lösung mit zwei zusätzlichen Argumenten der Funktion. 

function reverseDigits(num: number, reversed = 0, sign=1): number {

    if (num < 0) sign = -1;
    if (num === 0) return sign * reversed;
    num = Math.abs(num);
    
    return reverseDigits(Math.floor(num / 10), reversed * 10 + num % 10, sign);
}
