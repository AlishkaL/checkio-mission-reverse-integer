Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę znaleźć wartość bezwzględną danej liczby, zamienić ją na ciąg i odwrócić. Jeśli oryginalna liczba jest ujemna - proszę uzupełnić końcową liczbę minusem. Proszę przekonwertować wszystko z powrotem na liczbę całkowitą.

Ok, od czego powinienem zacząć?

Proszę znaleźć wartość bezwzględną danej liczby za pomocą funkcji abs(). 

abs(num)

Konwertowanie na ciąg znaków za pomocą funkcji str(). 

str(abs(num))

Proszę odwrócić ciąg znaków za pomocą funkcji negative slicing. 

str(abs(num))[::-1]
Co robić dalej?

jeśli oryginalna liczba jest ujemna, proszę uzupełnić wyrażenie za pomocą "-". W tym celu proszę wykorzystać zdolność Pythona do interpretowania wartości logicznej jako liczby całkowitej. 

'-' * (num < 0)
Nadal potrzebuję podpowiedzi.

Proszę połączyć dwa powyższe wyrażenia i przekonwertować wynik z powrotem na liczbę całkowitą. 

int('-' * (num < 0) + str(abs(num))[::-1])
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcji i zwrócić powyższe wyrażenie. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

def reverse_digits(num: int) -> int:

    return int('-' * (num < 0) + str(abs(num))[::-1])

Jasne, proszę łapać!

Bez użycia konwersji ciągów, proszę w sposób ciągły podzielić podaną liczbę przez 10 i utworzyć nową odwróconą liczbę. 

def reverse_digits(num: int) -> int:

    reversed_num, absolute_num = 0, abs(num)
    while absolute_num:
        absolute_num, remainder = divmod(absolute_num, 10)
        reversed_num = reversed_num * 10 + remainder
        
    return reversed_num if num>= 0 else -reversed_num

Rozwiązanie rekurencyjne z dwoma dodatkowymi argumentami funkcji. 

def reverse_digits(num: int, reversed_num=0, sign=1) -> int:

    if num < 0: sign = -1
    if num == 0: return reversed_num if sign> 0 else -reversed_num
    a, b = divmod(abs(num), 10)
    
    return reverse_digits(a, reversed_num * 10 + b, sign)
