Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę znaleźć wartość bezwzględną danej liczby, zamienić ją na ciąg i odwrócić. Jeśli oryginalna liczba jest ujemna - proszę uzupełnić końcową liczbę minusem. Proszę przekonwertować wszystko z powrotem na liczbę całkowitą.

Ok, od czego powinienem zacząć?

Proszę znaleźć wartość bezwzględną danej liczby za pomocą funkcji Math.abs(). 

Math.abs(num)

Proszę przekonwertować ją na ciąg znaków za pomocą metody toString() i podzielić za pomocą metody split() z pustym argumentem ciągu znaków. 

Math.abs(num).toString().split('')

Proszę odwrócić tablicę za pomocą metody reverse(). 

Math.abs(num).toString().split('').reverse()

Dołączenie tablicy z powrotem do łańcucha za pomocą metody join() z pustym argumentem łańcucha. 

Math.abs(num).toString().split('').reverse().join('')
Co robić dalej?

jeśli oryginalna liczba jest ujemna, proszę uzupełnić wyrażenie za pomocą "-". W tym celu proszę użyć operatora trójargumentowego. 

(num < 0? '-':'')
Nadal potrzebuję podpowiedzi.

Proszę połączyć dwa powyższe wyrażenia i przekonwertować wynik z powrotem na liczbę całkowitą. 

parseInt((num < 0? '-':'') + Math.abs(num).toString().split('').reverse().join(''))
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć swój kod w wyrażenie funkcyjne i zwrócić powyższe wyrażenie. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

function reverseDigits(num: number, reversed = 0, sign=1): number {

    return parseInt((num < 0? '-':'') + Math.abs(num).toString().split('').reverse().join(''))
}

Jasne, proszę pobrać!

Bez użycia konwersji ciągów, proszę w sposób ciągły podzielić podaną liczbę przez 10 i utworzyć nową odwróconą liczbę. 

function reverseDigits(num: number): number {

    let reversed = 0;
    const sign = num < 0? -1: 1;
    num = Math.abs(num);
    while (num!== 0) {
        reversed = reversed * 10 + num % 10;
        num = Math.floor(num / 10);
    }
    return sign * reversed;
}

Rozwiązanie rekurencyjne z dwoma dodatkowymi argumentami funkcji. 

function reverseDigits(num: number, reversed = 0, sign=1): number {

    if (num < 0) sign = -1;
    if (num === 0) return sign * reversed;
    num = Math.abs(num);
    
    return reverseDigits(Math.floor(num / 10), reversed * 10 + num % 10, sign);
}
